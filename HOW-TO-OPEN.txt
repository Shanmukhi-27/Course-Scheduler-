## If Something Goes Wrong:

### Error: "npm is not recognized"
- Node.js is not installed properly
- Go back to Step 1

### Error: "Cannot find module"
- Go back to Step 4 and run: npm install

### Port already in use:
- Close Command Prompt
- Open new Command Prompt
- Start from Step 3
